# Weather App

A simple Weather application which shows the weather for any city.

# Used Stacks

- Javascript
- Html
- CSS
- Webpack
- Semantic Ui

#Prerequisites

- NPM
- WEBPACK
- GIT

## Getting started

To get started with the app, clone the repo

```
git clone => The SSH key or URL <=
```

Change to Directory

```
cd  Weather-App
```

Run

```
npm install
```

Run

```
npx webpack --watch
```

open dist/index.html file on your browser and search for a city.
